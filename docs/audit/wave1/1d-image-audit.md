5. **Besaid chapter uses an S.S. Liki image.** image_0029_00 is from EPUB pg 29 (S.S. Liki content) but is assigned to two besaid sublocations. The besaid-specific image is only image_0027_00 (EPUB pg 27).

6. **image_0059_00 is assigned to 3 chapters** (via-purifico, bevelle, airship). Only the via-purifico assignment matches the EPUB origin (pg 59).

8. **image_0031_00 used twice within kilika** (Kilika Port + Kilika Woods) -- may be intentional if the guide map covers both areas, but worth verifying.
